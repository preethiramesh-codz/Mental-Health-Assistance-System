 Mental Health Monitoring and Wellness Recommendation System

 CodTech Internship Project

 Intern ID: CTTS191

 Overview
An NLP and Machine Learning based system that analyzes user text, predicts mental health categories, performs sentiment analysis, assesses risk level, calculates wellness score, and provides personalized recommendations.

 Features
- Mental State Prediction
- Sentiment Analysis
- Emotion Detection
- Risk Assessment
- Wellness Score Calculation
- Personalized Recommendations
- History Tracking
- Analytics Dashboard

 Technologies Used
- Python
- Streamlit
- Scikit-Learn
- TextBlob
- Pandas
- NLP

 Dataset
Mental Health Text Dataset from Kaggle.
